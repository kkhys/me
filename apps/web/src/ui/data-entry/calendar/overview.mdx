```jsx
'use client';

import * as React from 'react';

import { Calendar } from '#/ui/data-entry';

export const CalendarDemo = () => {
  const [date, setDate] = (React.useState < Date) | (undefined > new Date());

  return <Calendar mode='single' selected={date} onSelect={setDate} className='rounded-md border shadow' />;
};
```

## When To Use

When data is in the form of dates, such as schedules, timetables, prices calendar, lunar calendar.

## Anatomy

```jsx
import { Calendar } from '#/ui/data-entry';

export default () => <Calendar />;
```

## Usage

### Date Picker
