## When To Use

- A simple popup menu to provide extra information or operations.
- Comparing with `HoverCard`, besides information `Popover` card can also provide action elements like links and buttons.

## Anatomy

Import all parts and piece them together.

```jsx
import {
  Popover,
  PopoverContent,
  PopoverTrigger,
} from "#/ui/data-display";

export default () => <Popover>
  <PopoverTrigger />
  <PopoverContent />
</Popover>
```
