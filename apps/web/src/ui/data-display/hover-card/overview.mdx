## Anatomy

Import all parts and piece them together.

```jsx
import { HoverCard, HoverCardContent, HoverCardTrigger } from '#/ui/data-display';

export default () => (
  <HoverCard>
    <HoverCardTrigger />
    <HoverCardContent />
  </HoverCard>
);
```

## Examples

### Show instantly

Use the `openDelay` prop to control the time it takes for the hover card to open.

<Canvas of={HoverCardStories.ShowInstantly} layout='fullscreen' />

```jsx
import { HoverCard, HoverCardContent, HoverCardTrigger } from '#/ui/data-display';

export default () => (
  <HoverCard openDelay={0}>
    <HoverCardTrigger>…</HoverCardTrigger>
    <HoverCardContent>…</HoverCardContent>
  </HoverCard>
);
```
