## Anatomy

Import all parts and piece them together.

```jsx
import { Avatar, AvatarFallback, AvatarImage } from '#/ui/data-display';

export default () => (
  <Avatar>
    <AvatarImage />
    <AvatarFallback />
  </Avatar>
);
```
