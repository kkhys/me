## Props

<Canvas of={AvatarStories.Default} />

```jsx
import { Avatar, AvatarFallback, AvatarImage } from '#/ui/data-display';

export const AvatarDemo = () => (
  <Avatar>
    <AvatarImage src='https://github.com/kkhys.png' alt='@kkhys' />
    <AvatarFallback>KK</AvatarFallback>
  </Avatar>
);
```

## Usage

```jsx
import { Avatar, AvatarFallback, AvatarImage } from '#/ui/data-display';

<Avatar>
  <AvatarImage src='https://github.com/kkhys.png' />
  <AvatarFallback>KK</AvatarFallback>
</Avatar>;
```
