```jsx
import { Tooltip, TooltipContent, TooltipProvider, TooltipTrigger } from '#/ui/data-display';
import { Button } from '#/ui/general';

export const TooltipDemo = () => (
  <TooltipProvider>
    <Tooltip>
      <TooltipTrigger asChild>
        <Button variant='outline'>Hover</Button>
      </TooltipTrigger>
      <TooltipContent>
        <p>Add to library</p>
      </TooltipContent>
    </Tooltip>
  </TooltipProvider>
);
```

## When To Use

- The tip is shown on mouse enter, and is hidden on mouse leave. The Tooltip doesn't support complex text or operations.
- To provide an explanation of a `button/text/operation`. It's often used instead of the html `title` attribute.

## Anatomy

Import all parts and piece them together.

```jsx
import { Tooltip, TooltipContent, TooltipProvider, TooltipTrigger } from '#/ui/data-display';

export default () => (
  <TooltipProvider>
    <Tooltip>
      <TooltipTrigger />
      <TooltipContent />
    </Tooltip>
  </TooltipProvider>
);
```

## Examples

### Show instantly

Use the `delayDuration` prop to control the time it takes for the tooltip to open.

<Canvas of={TooltipStories.ShowInstantly} />

```jsx
import {
  Tooltip,
  TooltipContent,
  TooltipProvider,
  TooltipTrigger,
} from "#/ui/data-display";

export default () => <TooltipProvider>
  <Tooltip delayDuration={0}>>
    <TooltipTrigger>…</TooltipTrigger>
    <TooltipContent>…<TooltipContent>
  </Tooltip>
</TooltipProvider>
```
