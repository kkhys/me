## When To Use

To display a collection of structured data.

## Anatomy

Import all parts and piece them together.

```jsx
import { Table, TableBody, TableCaption, TableCell, TableHead, TableHeader, TableRow } from '#/ui/data-display';

export default () => (
  <Table>
    <TableCaption />
    <TableHeader>
      <TableRow>
        <TableHead />
      </TableRow>
    </TableHeader>
    <TableBody>
      <TableRow>
        <TableCell />
      </TableRow>
    </TableBody>
  </Table>
);
```

## Usage

### Data Table

You can use the `<Table />` component to build more complex data tables. Combine it with <a href='https://tanstack.com/table/v8' target='_blank' rel='noreferrer noopener'>@tanstack/react-table</a> to create tables with sorting, filtering and pagination.

See the <a href='/?path=/docs/data-display-data-table--overview'>Data Table</a> documentation for more information.

You can also see an example of a data table in the <a href='https://ui.shadcn.com/examples/tasks' target='_blank' rel='noreferrer noopener'>Tasks</a> demo.
