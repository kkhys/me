<Title>Badge</Title>
<Subtitle>Displays a badge or a component that looks like a badge.</Subtitle>

## When To Use

- It can be used to tag by dimension or property.
- When categorizing.

## Props

<Canvas of={BadgeStories.Default} />
<Controls of={BadgeStories.Default} />

```jsx
import { Badge } from '#/ui/data-display';

export const BadgeDemo = () => <Badge>Badge</Badge>;
```

## Usage

```jsx
import { Badge } from '#/ui/data-display';

<Badge variant='outline'>Badge</Badge>;
```

### Link

You can use the `badgeVariants` helper to create a link that looks like a badge.

```jsx
import Link from 'next/link';

import { badgeVariants } from '#/ui/data-display';

<Link href='/' className={badgeVariants({ variant: 'outline' })}>
  Badge
</Link>;
```
