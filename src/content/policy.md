当サイト（以下「本サイト」）は、Keisuke Hayashi（以下「運営者」）が運営しています。本プライバシーポリシーでは、本サイトをご利用いただく際の情報の収集、利用、および保護について説明します。

## 収集する情報

### アクセスログおよびアナリティクス

本サイトでは、利用状況を把握するために、プライバシーに配慮したセルフホスト型アナリティクスツール [Umami](https://umami.is/) を使用しています。Umami は Cookie を使用せず、個人データを収集せず、サイト間で利用者を追跡しません。以下の匿名化されたデータが収集されます。

- ページの URL およびリファラー
- ブラウザの種類およびオペレーティングシステム
- デバイスの種類
- おおよその地理的位置（匿名化された IP から導出される国レベル）

すべてのデータは集計された状態であり、個々の訪問者を特定することはできません。

### サーバーログ

本サイトは [Vercel](https://vercel.com/) でホスティングされています。Vercel は運用およびセキュリティ上の目的で、IP アドレス、ブラウザの種類、アクセス日時などの標準的なサーバーログ情報を収集する場合があります。詳細は Vercel の[プライバシーポリシー](https://vercel.com/legal/privacy-policy)をご参照ください。

## 第三者サービス

本サイトでは以下の第三者サービスを利用しています。

### 埋め込みコンテンツ

- YouTube — ブログ記事には [lite-youtube-embed](https://github.com/nicorevin/lite-youtube-embed) を介して YouTube 動画が埋め込まれている場合があります。埋め込み動画を操作すると、YouTube にデータが送信される場合があります。詳細は [Google のプライバシーポリシー](https://policies.google.com/privacy)をご参照ください。
- X（Twitter） — ブログ記事にはツイートが埋め込まれている場合があります。表示時に X にデータが送信される場合があります。詳細は [X のプライバシーポリシー](https://x.com/en/privacy)をご参照ください。

### API

- GitHub API — コントリビューションデータの表示に使用しています。訪問者の個人データは GitHub に送信されません。

## Cookie

本サイトでは、アナリティクスやトラッキングの目的で Cookie を使用していません。第三者の埋め込みコンテンツ（YouTube、X など）は、操作時に独自の Cookie を設定する場合があります。

## データの保持

本サイトでは個人データを保存していません。Umami で収集されたアナリティクスデータは、集計・匿名化された形式で保持されます。

## お子様のプライバシー

本サイトでは、お子様の個人情報を意図的に収集することはありません。

## 利用者の権利

本サイトでは個人データを収集していないため、アクセス、訂正、または削除の対象となる個人データは原則としてありません。ご不明な点がございましたら、運営者までお問い合わせください。

## ポリシーの変更

本ポリシーは予告なく変更される場合があります。変更後のポリシーは、本ページに掲載した時点で効力を生じます。
